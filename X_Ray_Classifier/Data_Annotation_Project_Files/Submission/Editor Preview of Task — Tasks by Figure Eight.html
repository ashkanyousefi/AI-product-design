Select The Normal X-Ray From The Pneumonia In A Data-Set Related To The Children Chest X-Ray

Instructions

Overview

In this task you will distinguish the normal x-ray image from the x-ray image which contains pneumonia. Then, you will use polygon or rectangle to select the part of the x-ray image which contains pneumonia.

In the next step, you will answer a question to determine your confidence level in selecting the pneumonia in the x-ray image.

In case you are not able to certainly identify if the x-ray image contains the pneumonia, just select others.

Steps

  1. Look over the provided image
  2. Determine if the image is a normal x-ray / contains pneumonia or you can select other if you can not identify
    • Refer to the help section to understand the difference between normal x-ray from the x-ray contains pneumonia
  3. Mark the pneumonia in the x-ray image using the provided polygon tool
  4. Select the confidence level in identifying the image status (normal/pneumonia/Other)
    • If you are sure that the image is normal x-ray we can select 10 for the confidence level and if we are not sure about the image being normal any number less than 10 could be selected depending on how certain you are about your selected choice.
    • If you select other the confidence scale will not be shown

Rules and Tips:

The x-ray image can be classified as normal, pneumonia and other.

  • A normal, healthy image will depict clear lungs without any areas of abnormal cloudiness/opacity; there may be structured, web-like vasculature in the lungs but otherwise that area should be clear. In healthy images, you are also more likely to see a diaphragm shadow as indicated in the following x-ray images.
  • A pneumonia image may include a few things: areas of cloudiness/opacity in several concentrated areas or one large area. You may also see a general pattern of opacity that obscures the structure of the lungs, heart, and diaphragm.
  • Other means that you are not able to identify if the x-ray image is normal or pneumonia
  • You can select the confidence level as well when you select normal or pneumonia

Steps for the process:

  • Make sure to use only one polygon or rectangle for each object
  • Make sure to draw polygon or rectangle to fully cover the area for the pneumonia
  • A normal, healthy image will depict clear lungs without any areas of abnormal cloudiness/opacity; there may be structured, web-like vasculature in the lungs but otherwise that area should be clear. In healthy images, you are also more likely to see a diaphragm shadow.

  • If not able to make a decision about an x-ray image, select the other option
